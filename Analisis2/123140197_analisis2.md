##  Langkah-langkah

### Step 1 — Membuat Area Proyek
Buat directori utama proyek dengan nama package dan masuk kedalamnya
```bash
cd ..; mkdir package; cd package
```
### Step 2 — Membuat File (Setup.py)
Buat file `setup.py` di dalam direktori `package/` dan masukkan kode berikut. 
```python
from setuptools import setup

# List of dependencies installed via 'pip install -e .'
requires = [
    'pyramid',
    'waitress',
]

setup(name='tutorial',
      install_requires=requires,
)
```
File ini mendefinisikan dependensi yang dibutuhkan proyek.

### Step 3 — Instalasi dan Persiapan Direktori Kode
```bash
$VENV/bin/pip install -e .
mkdir tutorial
```
Perintah `pip install -e .` memasang proyek dalam editable mode (mode pengembangan lokal). Artinya, setiap perubahan pada kode sumber akan langsung diterapkan tanpa perlu instal ulang. Kemudian, `mkdir tutorial` membuat direktori baru bernama tutorial yang berisi kode sumber aplikasi Pyramid.

### Step 4 — Membuat File `tutorial/__init__.py`
```bash
# package
```
File `__init__.py` digunakan agar Python mengenali folder tutorial sebagai sebuah package.
Walaupun kosong, file ini penting karena menandai direktori tersebut sebagai modul Python yang dapat diimpor.

### Step 5 — Membuat File `App.py`
```python
from waitress import serve
from pyramid.config import Configurator
from pyramid.response import Response


def hello_world(request):
    print('Incoming request')
    return Response('<body><h1>Hello World!</h1></body>')


if __name__ == '__main__':
    with Configurator() as config:
        config.add_route('hello', '/')
        config.add_view(hello_world, route_name='hello')
        app = config.make_wsgi_app()
    serve(app, host='0.0.0.0', port=6543)
```
Mengimpor librari dari code sebelumnya.

### Step 6 — Menjalankan Aplikasi WSGI
```bash
$VENV/bin/python tutorial/app.py
```
Perintah ini digunakan untuk menjalankan aplikasi Pyramid menggunakan Python yang ada di lingkungan virtual ($VENV).
Server akan aktif dan siap menerima permintaan (request) melalui browser.

### Step 7 — Membuka Aplikasi di Browser
Buka tautan berikut di browser:
`(http://localhost:6543/)`  
jika berhasi maka akan menampilkan `Hello world!`
