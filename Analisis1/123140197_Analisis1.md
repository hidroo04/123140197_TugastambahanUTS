## üìÑ Kode Hello World
```python
from waitress import serve
from pyramid.config import Configurator
from pyramid.response import Response


def hello_world(request):
    print('Incoming request')
    return Response('<body><h1>Hello World!</h1></body>')


if __name__ == '__main__':
    with Configurator() as config:
        config.add_route('hello', '/')
        config.add_view(hello_world, route_name='hello')
        app = config.make_wsgi_app()
    serve(app, host='0.0.0.0', port=6543)
```

## Analisis
1. The if __name__ == '__main__': is Python's way of saying, "Start here when running from the command line", rather than when this module is imported
   ```python
   if __name__ == '__main__':
   ```
   Blok diatas hanya akan di eksekusi saat file dijalankan secara langsung oleh app.py bukan saat fiile di import dari file lain. karena Blok if akan memastikan server hanya menyala ketika file ini adalah file utama yang dieksekusi.
2. Baris 12-14
   ```python
   with Configurator() as config:
        config.add_route('hello', '/')
        config.add_view(hello_world, route_name='hello')
   ```
   - ```python
     with Configurator() as config:
     ```
     baris blok code diatas adalah objeck yang digunakan untuk mengatur routing dan view dalam pyramid.
   - ```python
     config.add_route('hello', '/')
     ```
     blok ini merupakan langkah awal membuat route dengan memberi tahu pyramid bahwa akan ditambahkan sebuah URL (/) dengan nama route 'hello'.
   - ```python
     config.add_view(hello_world, route_name='hello')
     ```
     Langkah selanjutnya setelah membuat route, baris tersebut menghubungkan route dan view function. Saat pengguna mengakses alamat URL yang telah didefinisikan sebelumnya dengan `config.add_route('hello', '/')`, Pyramid akan mencari route bernama `hello'`, kemudian mengeksekusi fungsi `hello_world()` yang ditetapkan melalui baris `add_view` tersebut. Dengan kata lain, baris ini memberi tahu Pyramid bahwa setiap kali ada permintaan (request) ke URL `'/'`, maka sistem harus memanggil fungsi `hello_world` untuk menghasilkan respons ke browser.
3. Baris 6-8
   ```python
   def hello_world(request):
        print('Incoming request')
        return Response('<body><h1>Hello World!</h1></body>')
   ```
   - Baris `def hello_world(request):` mendefinisikan sebuah fungsi bernama `hello_world` yang menerima satu parameter bernama `request`. saat pengguna membuka halaman web, Pyramid akan mengirimkan objek request ini berisi informasi seperti alamat URL, metode (GET/POST), header, dan lain-lain. Fungsi ini nantinya akan digunakan sebagai view function, yaitu fungsi yang dijalankan setiap kali rute tertentu diakses.
   - Baris `print('Incoming request')` Baris ini digunakan untuk menampilkan pesan di terminal atau konsol setiap kali ada permintaan yang masuk ke server. setiap kali pengguna mengakses halaman `'/'`, server akan mencetak teks `"Incoming request"` di layar konsol, baris ini befungsi ssebagai debugging bagi pengembang.
   - Baris `return Response('<body><h1>Hello World!</h1></body>')` Baris ini me return objek `Response` dari `pyramid.response`. Objek ini berisi isi halaman web (body HTML) yang akan dikirimkan. Ketika browser menerima respons ini, ia akan menampilkan tulisan `‚ÄúHello World!‚Äù` di layar
4. Baris 15-16
   ```python
   app = config.make_wsgi_app()
   serve(app, host='0.0.0.0', port=6543)
   ```
   - Baris ` app = config.make_wsgi_app()` berfungsi untuk membangun sebuah web pyramid berdasarkan fungsi yang telah dibuat sebelumnya melalui `config`. sedangkan fungsi `make_wsgi_app()` mengubah seluruh pengaturan tersebut menjadi sebuah aplikasi yang sesuai dengan standar WSGI (Web Server Gateway Interface), sehingga dapat dijalankan oleh server web Python.
   - Baris `serve(app, host='0.0.0.0', port=6543)` digunakan untuk menjalankan aplikasi tersebut menggunakan Waitress. Parameter `host='0.0.0.0'` memungkinkan aplikasi diakses dari semua alamat IP dalam jaringan, sedangkan `port=6543` menentukan nomor port tempat aplikasi dijalankan.
